Take a roundabout and shrink up the printing hassle in Odoo!!

Everyday, numerous documents are printed in any business. Whether its a sales report or a customer receipt, printing documents is an integral part of any enterprise.

Normally you first download the document/report in your system, issue the print command, choose the format and send it to the printer. Repeating the process with every document is tedious and inefficient.

Also, downloading each and every report on a daily basis is hectic and takes system space too that in turn slows down your system too.

Odoo Document Printer via QZ Tray Module allows you to directly print the Odoo reports in Odoo. It facilitates cost-effectiveness and faster processing of the orders in the Odoo by eliminating the process of downloading these documents.

The module Print Reports Via Network/Thermal Printer cut-shorts the process and allows you to produce the hard copy of any printable report in Odoo with just a click.

Prerequisites

QZ Tray Client:

  • You need to install the QZ Tray client as the module uses the QZ Tray client for sending direct print commands to the printer. Follow the below link to install the QZ Tray Client- https://qz.io/download/

Noteworthy Points:

  • Make sure that Qz Tray Client is installed on your system before printing so it can connect to the network Printer.
  • The Odoo user needs to make sure that the name of the network printer entered in the created printer document record should exactly match the name of the printer connected to the network.
  • The QZ Tray Client uses the exact name to locate the Printer in the network and send the printing commands.

Calling QZ Tray For Printing:

  • Start the QZ Tray CLient on your system.
  • Now, open the form in Odoo you wish to print out. Click on the highlighted Print button on the top center of the form view.
  • It automatically calls the QZ Tray client to print the report on the basis of the configuration settings in the Odoo.

Network Printer Connection Status in Odoo

The Odoo backend shows the connection status of the set Network Printer to the user. The color of the printer icon indicates the current connection status of the printer.

  • Green: Connection Established

At Odoo backend, the QZ Tray Client automatically prompts the user to connect to the Network Printer. If the user allows the website to connect with the printer through the QZ Tray client and the connection is successfully established, the color of the icon turns green.

  • Yellow: Waiting For Printer Connection

The yellow printer icon indicates that the Odoo website is connected to the QZ Tray client but is not connected to the Network printer.

  • Red: Disconnected

If the printer icon in the Odoo website is red, it means the Odoo website was not able to connect with the specified printer. Make sure that QZ Tray Client is running on your system.

Detailed Features List


Print documents without downloading pdf in Odoo

  • Odoo Document Printer via QZ Tray Module allows you to directly print the Odoo reports such as sale, purchase, and invoice report/receipts, etc. in Odoo.
  • The Odoo plugin allows the Odoo user to directly print the Odoo reports using QZ Tray Client.


Print reports or receipts directly

  • The reports such as sale, purchase, and invoice report/receipts, etc are not needed to be downloaded as a PDF before printing them out.
  • The Odoo user can directly access the printer with the print button on the form view to print it.


Print the documents through Network/Thermal Printer in Odoo

  • The module currently supports Network/Thermal printers.
  • The Odoo user can also set a default Network/Thermal printer for the QZ Tray to use automatically on each reload. It is configurable.


Manage the configuration from the Odoo backend

  • Being the admin, you can enable/disable the synchronization of the printer with the website from the backend.

List of all created records for document printing.

Click on the print button to direct print the report/receipt of the current model
Connection status of the connected printer
Manage the printer auto sync configuration under settings option from the Odoo backend.
